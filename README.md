# Cobblemon Riding Tweaks

Configurable riding stamina and movement tweaks for Cobblemon.

Initial target:

- Minecraft 1.21.1
- Cobblemon 1.7.3
- Fabric first
- NeoForge layout prepared for follow-up support

The first gameplay feature will scale Cobblemon riding stamina drain without
touching Aprijuice, maximum stamina, serialization, or UI.
